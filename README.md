# 🌱 Agro Forte – Futuro Sustentável

O **Agro Forte** é um site institucional voltado ao agronegócio sustentável, unindo tecnologia, inovação e responsabilidade ambiental. O projeto apresenta informações sobre práticas sustentáveis, impacto ambiental e agricultura 4.0, com elementos interativos e animações.

---

## 🖥 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)  

---

## ✨ Funcionalidades

### 📊 Contadores Animados
- Animam automaticamente quando entram na tela.
- Usam `IntersectionObserver` para detectar visibilidade.
- Contam de 0 até o valor definido em `data-target`.

### ⬆️ Botão Voltar ao Topo
- Aparece ao rolar a página.
- Retorna suavemente ao topo ao ser clicado.

### 🧾 Formulário de Contato
- Valida campo de nome.
- Exibe mensagem de sucesso.
- Limpa os campos após envio.

### 🎯 Menu com Scroll Suave
- Navegação suave entre seções do site.
- Melhor experiência do usuário.

---

## 🎨 Design

- Tema baseado em sustentabilidade 🌿  
- Paleta de verdes e tons naturais  
- Layout responsivo com Grid e Flexbox  
- Cards informativos com ícones
